# CyberCut Studio

> Pro videó + villám web — kompromisszum nélkül.

Marketing weboldal videóvágáshoz, videógyártáshoz és weboldalkészítéshez. Sötét, cyberpunk stílusú design glassmorphism effektekkel és neon akcentusokkal.

## ✨ Funkciók

- **Sötét cyberpunk téma** — #0B0E14 háttér, neon lila/cián akcentusok
- **Glassmorphism kártyák** — backdrop-blur + neon border
- **Interaktív elemek** — Tab váltás, FAQ accordion, form validáció
- **Reszponzív design** — Mobil, tablet, desktop breakpointok
- **SEO optimalizált** — Meta tagek, structured data (LocalBusiness, Service)
- **Scroll animációk** — Intersection Observer alapú fade-in

## 📁 Projekt struktúra

## 🎨 Design System

| Változó | Érték | Használat |
|---------|-------|-----------|
| `--bg` | #0B0E14 | Fő háttér |
| `--surface` | #11162A | Kártya háttér |
| `--primary-neon` | #7C4DFF | Fő akcentus (lila) |
| `--secondary-neon` | #00E5FF | Másodlagos (cián) |
| `--accent-neon` | #00FF9A | Kiemelés (zöld) |

**Fontok:** Space Grotesk (címek) + Inter (szöveg)
